# análisis

Esta carpeta reúne materiales públicos de análisis y trazabilidad dentro del repositorio visible de Innova_N.

No debe leerse como archivo indiferenciado. A partir de esta fase se entiende como una carpeta madre con dos funciones:

- **auditorías**, cuando el objeto principal es revisar, contrastar o fijar un caso concreto;
- **análisis públicos**, cuando el objeto principal es producir una lectura estructural de utilidad pública desde el marco neodialéctico.
