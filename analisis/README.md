This folder gathers public analytical and traceability materials within Innova_N’s visible repository.

It should not be read as an undifferentiated archive. From this phase onward, it is understood as a parent folder with two functions:

- **audits**, when the main object is to review, contrast, or fix a specific case;
- **public analyses**, when the main object is to produce a structural public-utility reading from the neodialectical framework.
